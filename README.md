# dj_booking
# Призначення

Система призначена для обробки бронювань кімнат в готелі. Користувачі можуть переглядати доступні кімнати, реєструватися, увійти в систему, бронювати кімнати.

## Вимоги

- Python 3.x
- Django 3.x
- База даних (наприклад, SQLite, PostgreSQL)
